Porfolio

Memory Game

The Memory Game Project is all about demonstrating your mastery of JavaScript. This project is to build a complete browser-based card matching game. This isn’t just any memory game! It’s a snazzy, well-designed, feature-packed memory game.

Arcade game

The Arcade game is provided visual assets and a game loop engine; using these tools you must add a number of entities to the game including the player characters and enemies to recreate the classic arcade game frogger.

Restaurant Reviews

For the Restaurant Reviews projects, you will incrementally convert a static webpage to a mobile-ready web application. I will take a static design that lacks accessibility and convert the design to be responsive to different sized displays and accessible for screen reader use.

MyReads

In the MyReads project, I'll create a bookshelf app that allows you to select and categorize books you have read, are currently reading, or want to read. The project emphasizes using React to build the application and provides an API server and client library that you will use to persist information as you interact with the application.
